# GitHub Pages Deployment

Before deploying the `index.html` to GitHub pages, configure the repository settings:
- Go to the **Settings** under the repository
- Go to the **Pages** tab at the **Code, planning, and automation** group
- Select the **GitHub Actions** source under the **Build and deployment** section

Quicker alternative, navigate to the `https://github.com/<user>/<repo>/settings/pages` URL, then select build and deployment value.

Change the `index.html` to trigger the deployment.

See the **Actions** tab to inspect the workflows for more details.
